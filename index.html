Put them in their own tabs!

This is how it looks with applying the class .tabbed

It's very simple
Just give the class .tabbed to the parent element..
..and give each divs that needs to be tabbed a data-tabname attribute

For example, this div won't be tabbed because it doesn't have the data-tabname attribute.

Just remember to give each divs an unique data-tabname attribute.

Simple, lightweight, flexible!

As this plugin focuses primarily on getting a tabbed interface, it won't mess with each of the divs styling. You can still style them individually. For example, you can even apply bootstrap classes to the divs.


<div> data-tabname="Red div" class="bg-danger"></div>
<div> data-tabname="Green div" class="bg-success"></div>
<div> data-tabname="Blue div" class="bg-info"></div>
Custom header The tab header doesn't always have to be the same as the data-tabname attribute! You can define a custom one yourself. Check the following code that uses a fontawesome icon in the header.
<div> data-tabname="Div with custom tab header" class="bg-primary">
  <tabbed-custom-header><span class="fa fa-plus"></span></tabbed-custom-header>
</div>

How to style it?

In the CSS file, you will find the classes this plugin uses. You can style them however you want! (hint: there are only 5)
The default style can be changed to your heart content.
